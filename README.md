A movie browsing website built as part of the JavaScript 1 course assignment at the Frontend Development studies at Noroff.

## Description

This project was originally created as an introduction to JavaScript. For Portfolio 1, a new feature was added to improve the user experience and demonstrate state management.

Improvements made:
- Impemeneted a favourites feature using localStorage
- Stored favourites movie IDs for persistence across sessions
- Reused favourites logiv across list and product pages.
- Added accessible toggle states using "aria-pressed"

## Built with

- HTML
- CSS
- JavaScript (Vanilla)
